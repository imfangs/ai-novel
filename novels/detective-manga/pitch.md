# 名探漫剧

**经典刑侦剧 × 写实国风漫画 × 2集破案短剧**

---

## 一句话

> 3分钟破一案，用国风漫画复活中国最经典的探案故事。

## 项目概述

中国有三个探案IP是国民级的：**包拯、宋慈、狄仁杰**。

《少年包青天》《大宋提刑官》《神探狄仁杰》——这些剧加在一起，覆盖了从80后到00后的共同记忆。但原剧动辄40集，每案5-8集，节奏对今天的观众来说太慢了。

我们要做的事很简单：**把这些经典案件浓缩成2集漫剧短剧，每集3-5分钟，用写实国风漫画重新演绎。**

不是缩减版，是重新讲述。用漫画的视觉语言、电影级分镜、全程配音，让经典案件以一种全新的形态复活。

## 为什么是漫剧

### 漫剧的优势

**成本可控，质量上限高。** 不需要实拍的场地、演员、服化道成本，但可以实现实拍做不到的视觉效果——大宋的街市、阴森的密室、推理时的思维可视化。

**AI大幅降低制作门槛。** 写实国风的场景和角色，AI生图已经非常成熟。配合人工精修和分镜设计，小团队就能产出高质量内容。

**天然适合短视频平台。** 3-5分钟/集，竖屏友好，强悬念结构（第1集结尾留钩子），完美匹配抖音/小红书/视频号的算法偏好。

**漫画风格带来差异化。** 市场上的探案短剧大多是真人实拍（质感参差），或者纯动画（太卡通）。写实国风漫剧是一个空白位——既有氛围感，又有辨识度。

## 第一季：少年包青天

### 为什么从少年包青天开始

- **国民度**：2000年首播，豆瓣8.5+，一代人的童年
- **案件质量**：诡计精巧，有日式推理的水准
- **角色魅力**：包拯、公孙策、展昭、庞飞燕，辨识度极强
- **视觉适配**：宋代服饰、建筑天然适合国风漫画
- **零版权风险**：包拯是历史公共人物

### 选案（5案10集）

| # | 案件 | 核心诡计 | 为什么选 |
|---|------|----------|----------|
| 1 | 名扬天下 | 密室杀人+身份置换 | 包拯初登场，人物全亮相，知名度最高 |
| 2 | 血祭坛 | 密室+暗道+心理暗示 | 氛围感最强，漫画视觉冲击力大 |
| 3 | 隐逸村之谜 | 封闭空间+集体秘密 | "暴风雪山庄"模式，天然悬念 |
| 4 | 殿前扬威 | 宫廷阴谋+巧妙推理 | 场景升级到皇宫，画面恢弘 |
| 5 | 五鼠闹东京 | 连环布局+身份迷局 | 动作+推理双线，季终高潮 |

### 角色设定

| 角色 | 身份 | 漫画形象 |
|------|------|----------|
| **包拯** | 少年天才，面黑心正 | 黑面少年、眼神锐利、书生装扮但气场不凡 |
| **公孙策** | 包拯好友，聪明但骄傲 | 白衣公子、风度翩翩 |
| **展昭** | 御前侍卫，武功高强 | 利落束装、佩剑、英气逼人 |
| **庞飞燕** | 太师之女，古灵精怪 | 俏丽活泼、灵动 |

## 制作标准

### 画风：写实国风

- 写实风格，不Q版，不日漫
- 宋代考据：服饰、建筑、器物贴合历史
- 色调沉稳，案件高潮处冷色压暗
- 分镜追求电影感——大特写、景深、明暗对比

### 配音：全程真人配音

- 角色各有声线辨识度
- 旁白用"讲故事"的语气，不用解说腔
- 环境音效：街市、雨夜、衙门、推理时刻的标志性音效

### 单集节奏

**第1集·案发（3-5分钟）**
```
开场氛围 → 案件发生 → 初步调查 → 关键线索 → 悬念钩子
```
结尾字幕："你猜到凶手了吗？下集揭晓。"

**第2集·破案（3-5分钟）**
```
线索回顾 → 深入推理 → 真相揭露 → 动机揭示 → 余韵 + 下案预告
```

### 互动机制

- 第1集结尾放出"推理挑战"——观众在评论区猜凶手
- 第2集开头可展示"观众猜测TOP3"
- 每案画面中暗藏线索彩蛋，认真看的人可以提前猜出答案

## 受众画像

### 核心受众

**20-35岁，看过这些经典剧的一代人**
- 有童年情怀滤镜，看到"少年包青天"四个字就会点进来
- 已经不会再去看40集原剧了，但愿意花6分钟重温一个经典案件
- 活跃在抖音/小红书/B站

### 泛受众

**喜欢推理/探案内容的年轻人**
- 不需要看过原剧
- 被"3分钟破一案"的形式吸引
- 推理爱好者、国风爱好者

### 传播逻辑

每一案都是独立的传播单元——
- 情怀向："还记得少年包青天里最经典的这个案子吗？"
- 推理向："这个密室杀人你能在3分钟内猜到凶手吗？"
- 视觉向：国风漫画截图天然适合小红书传播

## 版权策略

| 元素 | 风险 | 策略 |
|------|------|------|
| 包拯/宋慈/狄仁杰 | 无 | 历史公共人物，自由使用 |
| 案件 | 低 | 基于历史典籍和民间传说重写，不照搬原剧 |
| 角色造型 | 无 | 全部原创设计，不模仿任何演员形象 |
| 台词 | 低 | 全部重新创作 |

**参考文献**：《三侠五义》《洗冤集录》《棠阴比事》《折狱龟鉴》等公共领域典籍。

## 执行计划

### Phase 1：概念验证（2-3周）
- 角色造型设计（AI生图+人工精修）
- 画风测试（产出5-10张概念图）
- 配音风格确定（试音）
- 第一案"名扬天下"脚本编写

### Phase 2：第一案制作（2-3周）
- 完成"名扬天下"2集漫剧
- 全流程跑通：脚本→分镜→画面→配音→后期
- 内部测试+调整

### Phase 3：发布与滚动制作（持续）
- 第一案发布，收集数据
- 根据反馈迭代
- 滚动制作后续4案
- 第一季10集全部完成后，启动第二季
