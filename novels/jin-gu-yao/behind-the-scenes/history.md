# 历史资料 — 平阳昭公主

## 一、正史记载

### 基本信息
- **身份**：唐高祖李渊第三女，嫡妻窦氏（窦皇后）所生
- **名字**：正史未记载真名（民间/影视常用"李秀宁""李三娘"，均非正史）
- **丈夫**：柴绍（隋元德太子千牛备身出身，后为凌烟阁二十四功臣第十四位）
- **儿子**：柴哲威、柴令武
- **谥号**：昭（"明德有功曰昭"）
- **下葬**：武德六年（623年）二月，以军礼下葬，中国封建史上唯一

### 核心事件时间线

| 时间 | 事件 |
|------|------|
| 大业十三年（617）五月 | 李渊决定起兵，密召柴绍夫妇 |
| 617年五月 | 柴绍赴太原，公主留长安自行安排 |
| 617年五月-八月 | 公主回鄠县李氏庄园，女扮男装自称"李公子"，散财赈民，招募数百人 |
| 617年六月-九月 | 收编何潘仁（手下数万人）、李仲文、向善志、丘师利等义军 |
| 617年间 | 连续攻占户县、周至、武功、始平等地 |
| 617年九月 | 李渊主力渡黄河入关中，公主已为其打下大片地盘 |
| 617年九月 | 柴绍带数百骑兵迎接公主 |
| 617年九月 | 公主挑选精兵一万余人，与李世民会师渭河北岸 |
| 617年十一月 | 夫妻各领一军（各自有幕府），共克长安 |
| 617年后 | 封"平阳公主"，因独有军功，赏赐异于他公主 |
| 617年后-623年 | 驻守娘子关（原名苇泽关），防守山西大本营 |
| 武德六年（623）二月 | 薨，以军礼下葬，谥"昭" |

### 关键史料原文

**《唐会要》记载：**
> "高祖第三女平阳公主。义兵起，公主于鄠县庄，散家资招引山中亡命，得数百人。起兵以应高祖。略地至盩厔武功始平，皆下之。每申明法令，禁兵无得侵掠，故远近奔赴甚众，得兵七万人。公主间使以闻，使者至，高祖大悦。及义军渡河。公主引精兵万馀，与太宗会於渭北，与其驸马柴绍，各置幕府。营中号为娘子军。京城平，封为平阳公主。以独有军功，每赏赐异於他主。及薨，追谥曰昭。"

**李渊反驳礼官（关于军礼下葬）：**
> "鼓吹，军乐也。往者公主于司竹举兵以应义旗，亲执金鼓，有克定之勋。周之文母，列于十乱；公主功参佐命，非常妇人之所匹也。何得无鼓吹！"

**《新唐书》评价：**
> "乃申法誓众，禁剽夺，远近咸附，勒兵七万，威振关中。"

### 柴绍离别时的对话
柴绍："你的父亲将要起兵扫平乱世，我打算前去迎接他的义旗，一起离开不可行，我独自走后又害怕你有危险，到底应该怎么办呢？"
公主："你应该赶紧离开，我是一个妇人，遇到危险容易躲藏起来，到那时自己会有办法的。"

## 二、重要人物

### 柴绍
- 出身将门，自幼"矫捷有勇力，以抑强扶弱"
- 曾为隋元德太子千牛备身
- 李渊起兵后，走小道赴太原，途中与李建成、李元吉汇合
- 授右领军大都督府长史，兼领马军总管
- 攻长安时与公主平级，各领一军，各有幕府
- 后随李世民征战：平薛举、破宋金刚、败王世充、擒窦建德
- 封霍国公，凌烟阁二十四功臣第十四位
- **有趣轶事**：被吐谷浑围困山谷时，临危不惧，安然而坐，让人弹琵琶、二美女翩翩对舞，吐谷浑兵放下弓矢观看，趁机派精骑绕后突袭，大败敌军
- 贞观十二年（638年）病逝

### 何潘仁
- 胡商（西域商人出身），手下有数万人
- 公主派家僮马三宝前去游说，何潘仁"居然甘愿做平阳公主的手下"
- 史书未详细记载马三宝用了什么手段 — **此处想象空间极大**

### 马三宝
- 公主家僮（家仆）
- 被派去游说何潘仁归降 — 一个家仆说服了一个拥兵数万的胡商，这本身就是传奇
- 后续事迹不详

### 李世民
- 公主的弟弟（一说兄弟，公主排行第三但具体与世民年龄关系不确定）
- 渡河后与公主会师渭北，共克长安

### 李渊
- 公主之父
- 对公主军功极为认可，力排众议以军礼下葬
- 评价公主"功参佐命，非常妇人之所匹也"

## 三、地理信息

### 鄠县（今陕西鄠邑区）
- 公主的起兵之地，李氏庄园所在
- 位于长安西南约50公里

### 关中地区攻占路线
- 户县 → 周至 → 武功 → 始平
- 由西向东推进，为李渊入关中扫清障碍

### 娘子关（原名苇泽关）
- 位于今山西省平定县东北
- 太行山上，出入山西的咽喉要道
- 因公主率"娘子军"驻守而改名
- 关城傍山近水，南侧群峰逶迤，北侧百丈悬崖下绵河湍急
- **注意**：有学者考证公主可能并未实际驻守此地，"娘子关"之名可能是后人附会。但作为小说可以合理使用。

### 渭河
- 公主与李世民会师之地（渭河北岸）

## 四、历史留白与想象空间

这些是正史没有记载、但对小说创作极有价值的空白：

1. **公主的真名** — 正史无载，我们可以赋予她一个名字
2. **柴绍离开后的心理** — 丈夫走了，她独自面对隋朝追捕，那一夜她在想什么？
3. **如何说服何潘仁** — 马三宝到底用了什么手段？一个家仆如何让拥兵数万的胡商臣服于一个贵族女性？
4. **军中日常** — 她如何管理七万人？如何建立军纪？如何处理士兵对女性统帅的质疑？
5. **两个儿子** — 何时出生？在她征战期间如何照顾？是否带在军中？
6. **与柴绍重逢** — 分别数月后，各自已是统帅，重逢时是什么场景？
7. **617年至623年的六年** — 攻克长安后到去世前的六年，史书几乎空白。她在做什么？为什么突然去世？
8. **死因** — 完全不详。病死？战死？难产？政治原因？想象空间巨大
9. **与李世民的关系** — 姐弟之间的互动，会师时的场景
10. **米汤退敌兵** — 野史轶事，可以作为素材

## 五、时代背景

### 隋末乱世（大业末年，约614-618年）
- 隋炀帝三征高句丽，国力耗竭
- 全国农民起义此起彼伏
- 三大主力：李密（瓦岗军）、窦建德、杜伏威
- 地方官吏纷纷割据称王
- 社会秩序崩溃，流民遍地
- 关中地区相对稳定但暗流涌动

### 李渊起兵
- 大业十三年（617年）五月，太原起兵
- 七月，统甲士三万晋阳誓师
- 九月，渡黄河入关中
- 十一月，攻克长安
- 武德元年（618年）五月，称帝建唐
